Για να κατασκευαστεί το jar στο IDE Intellij πηγαίνουμε στο File->Project Structure. Μετά
στα Artfacts. Πατάμε το κουμπί Add(+) έπειτα επιλέγουμε JAR->From modules with dependencies.Ως Main Class
βάζουμε την κλάση Main.Στο directory για META-INF βάζουμε τον φάκελο resources του project.Πατάμε ok και
πηγαίνουμε στην καρτέλα Build του Intellij.Στην συνέχεια επιλέγουμε Build Artifacts και στην καρτέλα Action
που θα εμφανιστεί διαλέγουμε Build.Το αρχείο jar τώρα βρίσκεται στον φάκελο out->artifacts->PAYDAY_CSD4613_jar
του project.

Προκειμένου να λειτουργήσει σωστά η λειτουργεία SAVEDATA LOADATA σιγουρευομαστε πως το Antivirus δεν εμποδίζει
το πρόγραμμα μας.Ωστόσο μέσω της κονσόλας, δηλ. με την εντολή java -jar PAYDAY_CSD4613.jar δεν δημιουργείται πρόβλημα στην
δημιουργεία SAVEDATA.Επομένως με το RUN.BAT διασφαλίζουμε ότι το Antivirus/Windows Defender θα μας επιτρέψει να δημιουργήσουμε
τα αρχεία για το SAVE και επομένως να τα φορτώσουμε αν χρειαστεί στην συνέχεια.
*NOTE 1* Τα αρχεία για το SAVEDATA μέσω commandline αποθηκεύονται/φορτώνονται στο/απο directory που βρισκόμαστε οπότε αν δεν
βλέπουμε αρχεία να δημιουργούνται στον φάκελο που βρίσκεται το εκτελέσιμο, σιγουρευόμαστε οτι το τρέχουμε απο εκείνο το directory.

*NOTE 2* Η λειτουργία LOAD SAVEDATA προυποθέτει ότι έχει γίνει αποθήκευση του παιχνιδιού προηγουμένως, αλλιώς δεν θα λειτουργήσει.

*NOTE 3* Το state του παιχνιδιού μπορεί να αποθηκευτεί μεσω του κουμπιού SAVEDATA άνω αριστερά του παραθύρου, ΜΟΝΟ ΣΤΗΝ ΑΡΧΗ ΕΝΟΣ ΓΥΡΟΥ
δηλαδή προτού κάποιος παίκτης πατήσει το κουμπί Roll Dice.Στην συνέχεια μήνυμα επιτυχίας θα εμφανιστεί και μπορούμε να κλείσουμε το παιχνιδι
και να το ξαναανοίξουμε πατώντας αυτή τη φορά LOAD SAVEDATA αντί για NEWGAME.

ΟΔΗΓΙΕΣ ΠΑΙΧΝΙΔΙΟΥ
1) Επιλέγουμε αν θα ξεκινήσουμε νέα παρτίδα ή αν θα συνεχίσουμε μία που αποθηκεύσαμε (βλ. NOTES)
2) Αν επιλέξουμε καινούρια, ορίζουμε την διάρκεια του παιχνιδιού σε μήνες.
3) Μήνυμα με το ποιος θα παίξει πρώτος θα εμφανιστεί στην οθόνη και στο πλαίσιο Infobox.
4) Ο παίκτης που παίζει πρώτος πατάει το κουμπί ROLL DICE
5) Στην συνέχεια το κουμπί MOVE! για να μετακινηθεί στο ταμπλό
6) Μόλις τελειώσει την σειρά του πατάει το κουμπί End Turn(Δίνονται οδηγίες για το τι πρέπει να κάνει στο πλαίσο Infobox).
